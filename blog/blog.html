Var

Scope basically refer to a place where varibale are workabel.When a var variable is declared outside of a function, the scope is global. This means that any variable declared outside of a function block using the var keyword is accessible throughout the entire window.When a variable is declared inside a function, it has function scope. This indicates that it is accessible and limited to that function.

Constant

We can use constant variable as in a scope. It is also immutable within a scope. We can use a constant variable for only declare a not re assigned value.
The scope is a scope is a constant variable is Block scope.
it can not be declare without initialization. It can not be update or re-declare.
It cannot be accessed without initialization, as it cannot be declared without initialization

Let

We can use a let variable in a scope. But like var let is immutable within its scope. We can not reassign the let varibale but we can use it for for same variable multiple value assign. Let variable can not work out of its scope. We can update let variable but can not reassign it.
Var variable scope is block scope. It can be declare with out declaration.

REST API

Representational State Transfer (REST) is an architectural style that defines a set of constraints to be used for creating web services. REST API is a way of accessing web services in a simple and flexible way without having any processing.
Working: A request is sent from client to server in the form of a web URL as HTTP GET or POST or PUT or DELETE request. After that, a response comes back from the server in the form of a resource which can be anything like HTML, XML, Image, or JSON. But now JSON is the most popular format being used in Web Services.
